# Mongoose Type Gen

A Cli tool for generating typescript definitions for mongoose schemas

### Supported features

- [x] Virtuals
- [x] Instance methods
- [x] Static methods
- [x] Timestamps
- [x] Nested Fields
- [x] Embedded Schemas
- [x] Required fields
- [ ] Custom data types
- [ ] Typings for method arguments and return types
